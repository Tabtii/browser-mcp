Quick Start

Running in 3 steps. No npm. No pip. No dependencies.

1. Install Extension

  1. Download from GitHub or Chrome Web Store
  2. Open chrome://extensions/
  3. Enable Developer mode (top right)
  4. Click Load unpacked → select the browser-mcp folder

2. Start Relay

python3 relay.py

Python 3 is pre-installed on macOS and Linux. On Windows, download from python.org.

3. Connect Your AI Agent

Add to your MCP client config (Cursor, Claude Desktop, etc.):

{
  "mcpServers": {
    "browser-mcp": {
      "command": "python3",
      "args": ["path/to/relay.py"]
    }
  }
}

Click the BrowserMCP extension icon → Start. Done!

Architecture

Minimal, local, transparent.

AI Agent (Claude / Cursor / Hermes)
    ↕ stdio (JSON-RPC)
relay.py (Python, zero deps)
    ↕ WebSocket (ws://127.0.0.1:9274)
Chrome Extension (offscreen document)
    ↕ chrome.scripting API
Browser Tab

All processing happens locally. No cloud. No telemetry. No external servers.
