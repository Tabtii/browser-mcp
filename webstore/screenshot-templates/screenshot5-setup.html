Setup in 3 Schritten

Kein npm. Kein Node.js. Kein pip. Nur Python 3.

1

🧩 Extension laden

chrome://extensions/ öffnen, Entwicklermodus aktivieren, Ordner laden.

# chrome://extensions/
→ Entwicklermodus: ON
→ Entpackt laden: browser-mcp/
2

🐍 Relay starten

Terminal öffnen, relay.py ausführen. Fertig.

# Terminal
python3 relay.py
# → ws://localhost:9274
3

🤖 AI-Agent verbinden

MCP-Config kopieren, in Cursor/Claude einfügen, Extension starten.

{ "mcpServers": {
"browser-mcp": {
"command": "python3",
"args": ["relay.py"]
}
}}

python3 relay.py
