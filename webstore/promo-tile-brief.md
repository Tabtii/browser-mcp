# BrowserMCP – Promo-Tile Design Brief

## Brand Identity
- Name: BrowserMCP
- Tagline: „AI Browser Control via MCP“
- Vibe: Tech, clean, developer-focused, trustworthy, fast
- Farbpalette: Dunkler Hintergrund (#0f172a), Akzent #3b82f6 (Blau), Weiß/Text #f8fafc
- Font: Moderne Sans-Serif (Inter, Roboto, SF Pro)

## Assets needed

### 1. Marquee Promo Tile
- Größe: 1400 × 560 px
- Inhalt: BrowserMCP Logo (B) + Text „BrowserMCP — AI Browser Control via MCP“ + kleiner Screenshot/Wireframe eines Browsers mit AI-Tools + Pro-Badge
- Stil: Dunkel, minimalistisch, leichte Glow-Effekte, viel Whitespace

### 2. Large Promo Tile
- Größe: 920 × 680 px
- Inhalt: ähnlich Marquee, aber kompakter. Fokus auf „35 Tools“ und „Zero Setup“
- Stil: Konsistent zu Marquee

### 3. Small Promo Tile
- Größe: 440 × 280 px
- Inhalt: Logo + „BrowserMCP“ + kurzer Subtext „AI agents control your browser“
- Stil: Konsistent, sehr reduziert

### 4. Screenshot 1 – Tool Grid
- Größe: 1280 × 800 px
- Inhalt: Landing Page / Popup mit allen 35 Tools als Grid + Pro-Badge
- Stil: Realistischer Chrome-Screenshot, dunkles Theme

### 5. Screenshot 2 – Agent in Action
- Größe: 1280 × 800 px
- Inhalt: Beispiel-Chat neben Browser-Fenster, das zeigt, wie ein Agent „Click Get Pro“ ausführt
- Stil: Split-Screen oder Overlay, clean

### 6. App Icon (bereits vorhanden)
- 128×128: `icons/icon128.png`
- 48×48: `icons/icon48.png`

## Design-Hinweise
- Kein Text zu klein (Chrome Store kann komprimieren)
- Kontrastreich
- Keine irreführenden „Play-Button“ oder „Download“-Pfeile
- Kein Wallet/Preis auf dem Tile (Pricing wird im Listing eingestellt)
- Pro-Badge dezent als gelber Stern oder grünes „Pro“-Label

## Ausgabeformate
- PNG mit transparentem oder dunklem Hintergrund
- Dateien benennen:
  - `marquee_1400x560.png`
  - `large_920x680.png`
  - `small_440x280.png`
  - `screenshot_1280x800_1.png`
  - `screenshot_1280x800_2.png`

## Ziel
Store-Besucher sollen innerhalb von 3 Sekunden verstehen: „BrowserMCP = AI Agent steuert meinen Browser.“
